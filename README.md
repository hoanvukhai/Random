# Chia Đội Bóng Ngẫu Nhiên ⚽

Ứng dụng web giúp chia đội bóng cực nhanh và công bằng. Không cần nhập tên từng người, chỉ cần xếp hàng, mỗi người bấm rút thăm một lần là có kết quả đội của mình.

## Tính năng
- 🌗 Hỗ trợ Giao diện Tối / Sáng tự động theo thiết bị.
- 💾 Lưu cài đặt tự động (localStorage) không sợ mất dữ liệu khi tải lại trang.
- 🎲 Thuật toán xáo trộn Fisher-Yates công bằng tuyệt đối.
- 📱 Tối ưu hóa hoàn toàn cho thiết bị di động.
- ⚡ Được xây dựng cực nhanh với Vite + React + Framer Motion.

## Cài đặt & Chạy cục bộ
1. Cài đặt các gói phụ thuộc: `npm install`
2. Chạy máy chủ phát triển: `npm run dev`
3. Mở trình duyệt ở địa chỉ `http://localhost:5173`
